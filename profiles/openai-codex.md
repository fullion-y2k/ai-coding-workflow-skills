# OpenAI Codex Profile

Allowed models:

- `gpt-5.5`
- `gpt-5.4`
- `gpt-5.4-mini`
- `gpt-5.3-codex`, only when available

All reasoning effort is `medium`.

Do not use `gpt-5.4-mini` as orchestrator for multi-step work. Use it for read-only exploration and small bounded implementation. Use `gpt-5.4` or `gpt-5.5` for orchestration and review-heavy decisions.
