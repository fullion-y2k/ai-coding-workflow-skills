# New Web App Heavy

## Example Request

Build a new authenticated business web app with roles, audit history, external billing integration, and admin workflows.

## Expected Route

Heavy.

## Human Gate

Confirm product requirements, auth boundaries, data model, external integration behavior, and acceptance criteria before implementation.

## Critical Review

Review requirements, design, implementation plan, diff, verification, rollback notes, and remaining risk.

## Browser Verification

Use browser tooling for main workflows when available. If unavailable, provide manual steps and mark browser verification as blocked/manual.
