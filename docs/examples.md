# Examples

## New Project

```text
$ai-new-project-delivery Build a small markdown notes app with search, import/export, tests, and browser verification.
```

Expected: Standard unless the scope expands into auth, external storage, or unclear product decisions.

## Existing Bug

```text
$ai-existing-project-change Fix the settings save failure after the latest refactor.
```

Expected: targeted investigation, cause analysis, minimal diff, regression verification.

## Heavy Change

```text
$ai-existing-project-change Add a new public API field, backfill existing records, and preserve old clients.
```

Expected: Heavy route, human gate, design, implementation plan, critical review, rollback notes.
